# App icon

The shipped icon is generated by `make_icon.py` — a four-segment neon donut
(matching the dashboard's category colours: cash · investments · pension ·
real estate) on a dark navy background with subtle corner glows.

## Regenerate the icon

```bash
pip3 install Pillow            # one-time
python3 tools/icon/make_icon.py
```

This overwrites `AssetRack/Assets.xcassets/AppIcon.appiconset/AppIcon.png` in
place. Commit the new PNG along with any script changes.

## Preview at home-screen size

```bash
python3 tools/icon/preview.py
open tools/icon/preview.png
```

Applies the iOS rounded-corner mask and renders on a dark backdrop so you can
see how the icon will look on the actual home screen.

## Iterating

Most knobs live at the top of `make_icon.py`:

| Knob | What it controls |
|---|---|
| `OUTER_F`, `INNER_F` | Donut radii (fractions of the canvas). Tweak for thicker / thinner ring. |
| `GAP_DEG` | Degrees of gap between segments. |
| `PROPS` | Per-segment size. `[0.25] * 4` = equal quarters; `[0.30, 0.32, 0.20, 0.18]` mimics a real allocation. |
| `DONUT_COLOURS` | The four segment colours, in clockwise order from 12 o'clock. |
| `BG_BASE` | Background colour. |
| `CORNER_GLOWS` | List of `(x_fraction, y_fraction, rgb)` glow sources. `(0,0)` = top-left. |
| `CORNER_RADIUS_F` | Glow reach. |
| `CORNER_INTENSITY` | Peak strength of the glow at the corner. |

After editing, regenerate + preview to see the change.
